# Sales Performance Dashboard Analysis

## Objective
Analyze sales data to identify:
- Revenue trends
- Top-performing products
- Regional performance
- Profit margins

## Tools Used
- SQL
- Excel
- Tableau

## KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Average Order Value

## Dashboard Features
- Sales Trend Chart
- Region-wise Sales Analysis
- Top Products
- Category Profit Analysis
- Filters

## Deliverables
- Dashboard File
- SQL Queries
